HTML TAG LIST

  1. <html>:the root element of an html Document
  2. <head>:contain the metadata of html Document
  3. <title>:sets the title of the html Document
  4. <meta>:provide addition information about the html Document
  5. <body>:Contains the visible content of the HTML document.
  6. <h1>, <h2>, <h3>, <h4>, <h5>, <h6>:Headings with varying levels of importance.
  7. <p>:Represent the paragraph of the text.
  8. <a>:Defines hyperlinks to other web pages or resources.
  9. <img>:embeds the image in html document
  10. <ul>:Creates an unordered list with bullet points.
  11. <ol>: Creates an ordered list with numbered items.
  12. <li>: Represents an item in a list, either unordered or ordered.
  13. <div>: A generic container element for grouping and styling purposes.
  14. <span>: A generic inline container element for styling specific portions of text or content.
